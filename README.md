# Installation : 
Clonez le projet :
```
$ git clone https://github.com/SarahHenriette/photosnap.git
```

Allez au repo cloné :
```
$ cd photosnap
```

Installez les packages npm (décrits dans `package.json`) :
```
$ npm install
```

Lancez l'application :
```
$ npm run serve
```

Puis allez à l'adresse : `http://127.0.0.1:8080/`
